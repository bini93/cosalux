# Cosalux Testproject

Testaufgabe von Cosalux

## Installation

Clone the repository:

```bash
git clone https://github.com/bini93/cosalux.git
```

Install dependencies:

```bash
yarn install
```

## Usage

Start the server:

```bash
yarn dev
```

The Link will be shown in your Terminal for example: [http://localhost:3000](http://localhost:3000/).
